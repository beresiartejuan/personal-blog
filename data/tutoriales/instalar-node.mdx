## Javascript para... ¿Computadoras?

Javascript es un lenguaje de programación increible, a estas alturas es casi indispensable cuando vamos a hacer una web y es uno de los lenguajes más populares. Sí eres nuevo en el mundo de la programación seguramente pensabas que Javascript solo se podía ejecutar en el navegador, pero tambíen se puede utilizar como un lenguaje de proposito general. ¡Al igual que Python!

Para ejecutar javascript fuera del navegador necesitas tener un interprete de Javascript. Sí buscas en internet como ejecutar Javascript en tu computadora seguramente te encontraras con **[NodeJS](https://nodejs.org/es/)**. Este es el programa más utilizado para ejecutar Javascript fuera del navegador y es el que vamos a instalar, sin embargo, hay que decir que tambíen existen alternativas a node: [Dino](https://deno.land/) y [Bun JS](https://bun.sh/). Dino es un moderno interprete de Javascript ( Y de TypeScript ), mientras que Bun es una alternativa que se centra en ser eficiente y más rapida que Node y Dino. Muy pronto voy hacer un blog contando las diferencias entre estos tres a profundidad.
